Municipal Dashboards is an independent project — it is not run by, or affiliated with, any city government. Every dashboard links back to the official city events calendar and to the original agenda PDFs for anything authoritative.

Built with a Claude agent pipeline (Haiku for PDF transcription, Sonnet for classification) on top of city-specific scrapers. Source on GitHub.

Want your city listed here? Each city is a small Python adapter plus a branding file. See the architecture notes to add one.
